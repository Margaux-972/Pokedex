# Pokédex App

A web application inspired by the Pokédex, allowing users to browse Pokémon and types, and access detailed information through interactive cards.

## ✨ Overview

Pokédex App is a dynamic web application that displays Pokémon data and their types. Users can navigate between pages, explore Pokémon by type, and view detailed information by interacting with cards.

## 🚀 Key Features

- 🐾 Pokémon Page
  - Browse a list of Pokémon
  - Displayed as interactive cards
- 🧬 Types Page
  - View all Pokémon types
  - Click on a type to display all Pokémon belonging to that type
- 🔍 Detailed Pokémon View
  - Click on a Pokémon card to access its details
  - Displays:
    - Name
    - Image
    - Type(s)

## 🛠️ Tech Stack

### Frontend

- React
- Axios

## API

- PokéAPI

## 🧩 How It Works

- Data is fetched from the PokéAPI
- The app displays:
  - A list of Pokémon
  - A list of types

## 🧬 Types Flow

1. User selects a type
2. The app fetches and displays all Pokémon with that type
3. Pokémon are shown as cards
4. User clicks on a Pokémon card
5. A detail view opens with:
   - Name
   - Image
   - Type(s)
